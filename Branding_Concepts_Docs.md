# Hera Branding & Logo Concepts

A branding foundation for Hera — identity, motifs, colors, and logo directions.

---

## Positioning Statement

**Hera – Cloud-agnostic Kubernetes for startups.**

Tone:

- Trustworthy
- Stable
- Calm
- Technical
- Not flashy
- Confidence in clarity and simplicity

---

## Logo Concepts

### Concept A – "Temple + Cloud + Cluster"

Symbolism:

- Greek temple: foundation, stability, origin of the name “Hera”
- Soft cloud shape: cloud providers
- Subtle cluster hexagon/dots: Kubernetes

Visual style:

- Flat minimal icon
- Works in one color
- Scalable to favicon size

---

### Concept B – "H Monogram as a Network"

Symbolism:

- “H” formed using rails + node connections
- Represents networks, clusters, relationships
- Halo or orbit: protection / platform guardian

Visual style:

- Monoline
- Simple geometry
- Excellent for CLI ASCII/logos

---

## Color Palette

Primary colors:

- **Deep blue / slate**: `#1E293B`
- **Royal/navy**: `#1D3557`

Accent colors:

- **Teal**: `#14B8A6`
- **Cyan**: `#0EA5E9`

Neutrals:

- Light gray or white backgrounds

Reasons:

- Feels technical and modern
- Not Kubernetes-purple
- Works on light and dark modes

---

## Typography

- Logo/title: **Poppins** or **Inter (Bold)**
- Body text: **Inter**, **Roboto**, or system sans fonts
- Large spacing, breathable, calm documentation style

---

## Diagram Style Guidelines

- Use columns and beams to metaphorically represent layers
- Rounded rectangles for modules
- Simple node/edge icons for cluster relationships
- Consistent use of Hera accent colors

- “Hera – Startup-ready Kubernetes.”
- “Hera – Infrastructure clarity for builders.”
